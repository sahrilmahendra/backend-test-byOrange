## About The Project

Berikut fitur yang terdapat dalam project ini :
| Feature | Authentication | Endpoint | Http Method
|---|---|---|---|
| Login user | No | localhost:8080/login | POST |
| Create user | No | localhost:8080/users | POST |
| Get all users | Yes | localhost:8080/users | GET |
| Get user by id | Yes | localhost:8080/users/:id | GET |
| Update user by id | Yes | localhost:8080/users/:id | PUT |
| Delete user by id | Yes | localhost:8080/users/:id | DELETE |
|---|---|---|---|
| Create item | Yes | localhost:8080/items | POST |
| Get all items | No | localhost:8080/items | GET |
| Get item by id | No | localhost:8080/items/:id | GET |
| Update item by id | Yes | localhost:8080/items/:id | PUT |
| Delete item by id | Yes | localhost:8080/items/:id | DELETE |
|---|---|---|---|
| Create order | No | localhost:8080/order | POST |
| Get all order | No | localhost:8080/order | GET |
